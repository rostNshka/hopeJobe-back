## 📝 hopeJobe-back
Часть Backend. \
Данный проект разработан исключительно в целях изучения и подкрепления знаний о библиотеках React.
Проект hopeJobe создан поиска работы и подбора персонала. Здесь представлена страница вакансий, избранные вакансии для
соискателя и возможность добавления вакансии работодателем. Так же в проекте представлен личный кабинет.

PS Данная часть проекта будет одной из двух. Вторая часть посвящена беккнд(серверной) разработке.

⚙️Технологии hopeJobe(Backend): \
node.js, \
express, \
Prisma, \
PostgresSQL, \
npm \

## 🚀 Использование:
Требования: Для установки и запуска проекта, необходим NodeJS v8+.

Установка зависимостей
Для установки зависимостей, выполните команду: \
`$ npm i`

Запуск приложения: \
`$ npm run dev`

Создание билда, чтобы выполнить production сборку, выполните команду:

`$ npm run build`
